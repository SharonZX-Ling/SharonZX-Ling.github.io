1修改前的准备

你需要一个代码编辑器来打开和编辑文件。推荐：

  • VS Code（免费，最推荐）→ 下载地址
  • 或者直接用记事本 / Notepad也能改（但不推荐，没有代码高亮）

安装好编辑器后，用 VS Code 打开整个项目文件夹 D:\Project\SharonZX-Ling.github.io

💡 快捷方式：在项目文件夹空白处右键 →「Open with Code」即可用 VS Code 打开整个文件夹。

2认识文件结构

你只需要关心以下几个文件：

文件 / 文件夹作用你需要改吗？
index.html所有文字内容都在这里✅ 主要改这个
css/style.css样式（颜色、大小、间距）一般不用改
js/main.js交互逻辑（弹窗、滚动）不用改
assets/images/存放图片文件✅ 放图片到这里
assets/pdfs/存放 PDF 文件✅ 放 PDF 到这里
assets/videos/存放本地视频文件可选
核心原则：90% 的修改只需要动 index.html 这一个文件。图片和 PDF 是放到 assets/ 文件夹里，然后在 HTML 里引用路径。

4替换 / 添加图片

4.1 把图片放到项目中

把你的图片文件复制到 assets/images/ 文件夹里。支持的格式：jpg、png、webp、svg。

⚠️ 文件名不要用中文，建议用英文+数字，如 brand-design.jpg、social-media-1.png

4.2 替换项目卡片中的图片

在 index.html 中搜索 image-cover，找到项目图片卡片：

index.html · Projects 板块 <article class="card project-card" data-type="image"> <div class="card-cover image-cover" data-src="assets/images/work-1.jpg" style="background-image:url('assets/images/work-1.jpg')"> <span class="badge badge-image">图片</span> </div> <div class="card-body"> <h3 class="card-title">品牌视觉设计</h3> <p class="card-desc">某新消费品牌从0到1的视觉体系搭建。</p> <button class="btn btn-ghost btn-open" data-type="image" data-src="assets/images/work-1.jpg">查看图片</button> </div> </article>

需要改 3 个地方（上图中高亮的部分）：

  1. data-src="assets/images/work-1.jpg" → 你的图片文件名
  2. style="background-image:url('assets/images/work-1.jpg')" → 同一个图片文件名
  3. data-src="assets/images/work-1.jpg"（按钮上的） → 同一个图片文件名
三处路径必须一致！第一处是点击卡片封面打开弹窗用的，第二处是显示缩略图用的，第三处是点击"查看图片"按钮打开弹窗用的。

4.3 添加一张新的图片作品

复制一个现有的图片卡片，粘贴在后面，修改内容即可：

复制这段，改3处路径 + 标题 + 描述 <article class="card project-card" data-type="image"> <div class="card-cover image-cover" data-src="assets/images/你的图片名.jpg" style="background-image:url('assets/images/你的图片名.jpg')"> <span class="badge badge-image">图片</span> </div> <div class="card-body"> <h3 class="card-title">作品标题</h3> <p class="card-desc">作品描述文字。</p> <button class="btn btn-ghost btn-open" data-type="image" data-src="assets/images/你的图片名.jpg">查看图片</button> </div> </article>

5添加视频作品

5.1 方式一：B站视频（推荐）

你的视频上传到 B 站后，获取嵌入链接。步骤：

  1. 打开你的 B 站视频播放页
  2. 点击视频下方「分享」→「嵌入代码」
  3. 复制 src="..." 里面的链接，类似：//player.bilibili.com/player.html?bvid=BV1xx411c7mD
  4. 在前面加上 https:，变成完整链接
index.html · 视频项目卡片 <article class="card project-card" data-type="video"> <div class="card-cover video-cover" data-src="https://player.bilibili.com/player.html?bvid=BV1xx411c7mD"> <div class="play-icon">▶</div> <span class="badge badge-video">视频</span> </div> <div class="card-body"> <h3 class="card-title">视频标题</h3> <p class="card-desc">视频描述。</p> <button class="btn btn-ghost btn-open" data-type="video" data-src="https://player.bilibili.com/player.html?bvid=BV1xx411c7mD">观看作品</button> </div> </article>
注意：data-src 出现 2 次，封面一次、按钮一次，两处链接必须相同。

5.2 方式二：本地视频文件

把视频放到 assets/videos/ 文件夹，然后链接写相对路径：

<!-- 只需把 data-src 改成本地路径 --> data-src="assets/videos/my-video.mp4"
⚠️ GitHub Pages 单文件限制 25MB。如果你的视频超过 25MB，建议上传 B 站 / YouTube 用外链方式。

5.3 方式三：YouTube 视频

YouTube 嵌入链接格式：

data-src="https://www.youtube.com/embed/视频ID" <!-- 视频ID 在 YouTube 视频地址 v= 后面那串字符 -->

8管理项目作品卡片

项目卡片有 3 种类型，区别在于 data-type 的值：

类型data-type弹窗行为
视频video弹窗用 iframe 播放（B站/YT）或 video 标签（mp4）
PDFpdf弹窗用 iframe 预览 PDF
图片image弹窗显示大图

添加新项目的方法：从上面 4-6 节找到对应类型的模板，复制粘贴到 <div class="card-grid two-col"> 里面即可。

💡 项目区域是 两列布局，卡片会自动从左到右、从上到下排列，不需要手动控制位置。

11本地预览（改完先看效果再上线）

修改完文件后，在浏览器里先预览确认效果：

方法一：直接打开

双击 index.html 文件，它会用默认浏览器打开。但部分功能（PDF 预览）可能无法显示，因为浏览器安全策略限制了本地文件访问。

方法二：启动本地服务器（推荐）

在项目文件夹打开终端（VS Code 菜单 → Terminal → New Terminal），运行：

终端命令 python -m http.server 8080

然后浏览器访问 http://localhost:8080 即可。所有功能（包括 PDF 预览）都能正常工作。

💡 按 Ctrl+C 可以停止服务器。修改文件后刷新浏览器即可看到更新。

12推送到 GitHub 上线

改完确认没问题后，推送到 GitHub：

方法一：让我帮你 push

直接在对话里说"帮我 push"，我会帮你提交并推送到 GitHub。

方法二：自己用命令行 push

在项目文件夹打开终端，依次运行 # 1. 添加所有改动 git add . # 2. 提交，引号里写这次改了什么 git commit -m "update: 更新实习经历和项目内容" # 3. 推送到 GitHub git push origin main
推送后多久上线？通常 1-2 分钟后 GitHub Pages 会自动构建完成。访问 https://SharonZX-Ling.github.io/ 刷新查看。如果没变化，按 Ctrl+Shift+R 强制刷新清缓存。

13常见问题

Q: 图片显示不出来 / 显示空白？

检查 3 点：

  1. 图片确实放在 assets/images/ 文件夹里了
  2. HTML 里的文件名和实际文件名完全一致（包括大小写、后缀名）
  3. 文件名没有中文、没有空格

Q: B站视频弹窗打不开 / 白屏？

B站的嵌入链接需要以 https://player.bilibili.com/player.html?bvid= 开头。确认你复制的链接格式正确。另外部分浏览器可能拦截 iframe，检查是否有弹窗拦截提示。

Q: PDF 预览打不开？

本地直接打开 HTML 时 PDF 可能无法预览（浏览器安全限制）。用本地服务器方式预览即可（见第 11 节）。线上 GitHub Pages 不受此限制。

Q: 改完之后网站没变化？

  1. 确认你已经 git push 了
  2. 等 1-2 分钟让 GitHub Pages 构建完成
  3. 浏览器 Ctrl+Shift+R 强制刷新
  4. 还不行就去 GitHub Actions 看有没有构建报错

Q: 我改错了，想撤销怎么办？

在 VS Code 里按 Ctrl+Z 撤销编辑。如果已经 push 了，告诉我"帮我回退到之前的版本"，我可以帮你用 git 回退。

Q: 想改颜色 / 字体 / 布局怎么办？

这些在 css/style.css 里。比如改主题色，搜索 --accent：

css/style.css · 第18行 --accent: #E94560; /* 改成你喜欢的颜色，如蓝色 #3B82F6 */
